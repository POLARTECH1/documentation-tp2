# GUIDE D’UTILISATION LIBREOFFICE WRITER
## 1. CRÉER UN NOUVEAU DOCUMENT
### 1.1 Mise en contexte

Cette fonctionnalité permet de créer un document texte vierge dans LibreOffice Writer.

### 1.2 Prérequis

LibreOffice installé sur le poste.

### 1.3 Étapes

1- Ouvrir LibreOffice.

2- Sélectionner Writer dans l’écran d’accueil.

3- Un document vierge s’ouvre automatiquement.

### 1.4 Résultat attendu

Un document texte vide est affiché à l’écran.

### 1.5 Erreur fréquente

Ouvrir un autre module (Calc ou Impress) par erreur.

## 2. ENREGISTRER UN DOCUMENT
### 2.1 Mise en contexte

L’enregistrement permet de conserver le document sur le poste afin de le réutiliser ultérieurement.

### 2.2 Prérequis

Un document Writer ouvert.

### 2.3 Étapes

1- Cliquer sur Fichier.

2- Sélectionner Enregistrer ou Enregistrer sous.

3- Entrer un nom de fichier.

4- Choisir un emplacement.

5- Cliquer sur Enregistrer.

### 2.4 Résultat attendu

Le document est enregistré et peut être rouvert ultérieurement.

### 2.5 Erreurs fréquentes

- Ne pas vérifier l’emplacement du fichier.

- Confondre Enregistrer et Enregistrer sous.

## 3. DIFFÉRENCE ENTRE "ENREGISTRER" ET "ENREGISTRER SOUS"

- Enregistrer : Met à jour le fichier existant.

- Enregistrer sous : Crée une nouvelle copie avec un autre nom ou emplacement.
